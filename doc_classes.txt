
World(world.py) = основной класс игры.
    Сохраняет основные параметры, номер игры, хода, год, дату создания игры.
    Список династий и игроков(id)
    Тут же сохраняем цену построек, высчитывается в конце хода.


Dynasty(dynasty.py) = один из основных классов, династия управляемая игроком.

Settlement = Поселение игрока.

Decision =  Решения игрока. Будет иметь набор методов, количество принятых и их цену.

Cities(cities.py) = Тут можно разместить нейтральные поселения для взаимодействия с ними.
Events(events.py) = События.
